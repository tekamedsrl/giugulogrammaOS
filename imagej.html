Realtime acquisition plugin

  • Directional keys to center the IJV
  • x to increase IJV wall treshold
  • z to decrease IJV wall treshold
  • s to increase detection radius
  • a to increase detection radius
  • p freez/start image to make image measurement
  • r start/stop JVP acquisition
  • e start/stop ECG acquisition
  • u upload acquired data

Acquisition procedure

  • In a PC open a window and manage to captures Eco images in realtime. Move the window to the top left corner fo the screen.
  • Start ImageJ and create a new Image 500x500 named JVP
  • Launch TM_RTJVP
  • Freez the Eco acquisition and pause the TM_RTJVP using p key
  • Measure 1 cm of the Freeze Eco image graduate scale using ImageJ tools
  • Open parameter window pressing t key
  • Set pixel per cm
  • Start the acquisition pressing p again then star the echography too
  • Click at center of the internal jugualr vein
  • Adjust grey level treshold using z and x. Adjust the scan ray length using a and s keys.
  • Make that ecg red cursor be on the bottom right corner
  • Start diagram recording using r and stop using r again
  • Upload data to the server using u kay. Set StudyID. Set repository name accordling with existing repository. Press OK to upload.
